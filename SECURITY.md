# Security Policy

## Supported Versions

Security fixes will be made against the default branch first.

## Reporting

If you believe you have found a security issue in this repository:

- do not open a public issue with exploit details
- contact the maintainer privately first
- include reproduction steps, affected versions, and any suggested mitigation

## Notes

This project handles flow artifacts and can validate against live NiFi APIs.
Be careful not to include secrets, bearer tokens, or customer flow data in issues,
logs, screenshots, or pull requests.
