Secrets and parameters

Preserve references, rehydrate values where they belong.

The advisor helps preserve flow structure and references. It should never try to become a secret recovery mechanism for values Apache NiFi did not export.

Back to site Open source doc

What the tool preserves

Exported structure, non-sensitive properties, component references, and Parameter Context wiring stay intact unless an explicit deterministic rewrite changes them.

What the tool cannot restore

Sensitive values that NiFi never exported remain an environment concern. Rehydrate them through Parameter Providers, Vault, Kubernetes Secrets, or other local systems.
